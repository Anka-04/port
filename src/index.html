Про меня

Меня зовут Анна

С 2011г занимаюсь развитием и продвижением сайтов. Сначала как менеджер по привлечению новых рекламодателей на портале Rabota.ru. Затем контент-менеджер в ГК Рубль Бум, сайт 1b.ru, позже как SMM-менеджер.
В 2021г начала обучение web-разработке. Создаю адаптивные сайты, использую библиотеку Bootstrap, технологии FlexBox и Grid, методологию БЭМ. Пользуюсь препроцессором Sass и программой Gulp.
Креативно подхожу к решению задач, инициативная, ответственная и дипломатичная.

Навыки

Что я использую в работе
HTML5
Именно он создает каркас вашего сайта или приложения, а пятая версия позволяет создавать еще более SEO-оптимизированную структуру вашего продукта
CSS3
Этот язык стилей позволяет создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!
Sass
Это препроцессор CSS, основной задачей которого является расширение возможностей написания CSS кода, препроцессор SASS ускоряет написание кода CSS и облегчает его обслуживание.
Gulp
Это инструмент, который помогает автоматизировать рутинные задачи веб-разработки.
Bootstrap
Используется для быстрой вёрстки адаптивных дизайнов сайтов и веб-приложений.
Visual Studio Code
Редактор кода
Figma
Это графический онлайн-редактор для совместной работы. В нём можно создавать прототипы сайта
